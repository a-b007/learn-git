1. git init                                       -> to initialise the folder for git
2. git remote add origin <repo-link>              -> (to link the local repo to a remote repo)
3. git add <file-name> or git add .               -> (to add a specific file or all files i.e. stage changes)
4. git commit -m "<commit-name>"                  -> (commit all changes that were staged/added)
5. git push -u origin master                      -> (push to origin repo master branch)
6. git branch                                     -> (check current git branch)
7. git branch <branch-name>                       -> (create a branch)
8. git checkout <branch-name>                     -> (go to the branch with name given)
9. git checkout master                            -> (go to master branch)
10. git push origin <branch-name>                 -> (push a branch to origin)
11. git merge <branch-name>                       -> (merge a branch with the current branch)
12. git branch -d sample-branch                   -> (delete a branch locally)
13. git push origin --delete sample-branch        -> (delete a branch from github remote)
